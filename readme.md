## 🙏 Commit 규칙 

```jsx
# 소문자로 작성!
git commit -m "feat/갤러리 무한스크롤 기능 구현"
```

|  | 유형 | 설명 |
| ------ | ------ | ------ |
| 🚀 | feat | 새로운 기능 추가, 수정 |
| 🐞 | fix | 버그 픽스, 에러 핸들링 |
| 🎉 | build | 빌드 & 배포 |
| 🔨 | chore | 패키지 외 기타 수정 |
| 📝 | docs | 문서 수정 |
| 🧹 | style | 코드 스타일, 포맷팅 변경 |
| 😎 | refactor | 리팩토링, 코드 개선 |
| 🙈 | chore | 짜잘한 수정 (주석) |
| 🚪 | release | 버전 릴리즈 |
| 🥪 | merge | 파일 병합 |
